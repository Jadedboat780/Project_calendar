# Project calendar
### Проект сдлеан совместно с [Prog-Maksim](https://github.com/Prog-Maksim) и [CaptainSliva](https://github.com/CaptainSliva). 
#### Это проект для нашего колледжа. Данная программа собирает данные о расписании с сайта колледжа и из Google Sheets. После чего она заносит эти данные в Google календарь.
## Как работает программа?
#### С понедельника по субботу в опредённое время программа запускает скачивание xlsx таблицы за сегодняшний день, после чего проверяет заполненность таблицы. Затем происходит поиск и удаление эвентов за сегодняшний день. Потом данные из таблицы заносятся в календарь в виде новых эвентов. Запуск этих процессов происходит в конце каждой парой.
#### В воскресенье  запускается процесс поиска и удаления эвентов за будущую неделю. Затем происходит парсинг расписания сайта колледжа для групп и преподавателей, а после это расписание заносится в календарь.
#### Основа проекта разделена на два ключевых фвайла: main.py и Schedule_Offices.py. Это сделано для ускарения работы удаления и создания эвентов(в будущем мы хотим разделить программу на три ключевых файла)
#### Также у нас есть телеграм-бот (@RKSI_Calendar_bot), который выдаёт расписание проеподователям и студентам.
## Файлы
+ #### main.py - файл с программой календаря, который создаёт расписание группам и преподавателям
+ #### id.py - файл с id-шниками календарей групп и преподавателей
+ #### Schedule_Offices.py - файл с программой календаря, который создаёт заполненность кабинетов
+ #### id_offices.py - файл с id-шниками кабинетов
+ #### RKSI_bot - папка с файлами телеграм бота
  + #### Aiogram_bot.py - файл бота
  + #### id.py - файл с id-шниками календарей групп, преподавателей и кабинетов
  + #### TEST_VIDEO.mp4 - видео с инструкцией
+ #### TESTS.json - файл с данными для работы с календарями(данный файл добавлен в .gitignore для конфиденциальности)
